border-radius

Bota arredondamento dos cantos de um elemento.

Exemplo: .card {   border-radius: 10px; }

background-color

Define a cor de fundo de um elemento.

Exemplo: .card {   background-color: lightblue; }

margin

Define o espaçamento externo de um elemento em relação aos outros.

Exemplo: .card h2 {   margin: 20px; }

font-size

Define o tamanho da fonte do texto em um elemento.

Exemplo: .h1 {   font-size: 18px; }

text-align

Define o alinhamento horizontal do texto, se vai começar na esquerda direita etc....

Exemplo: .h4 {   text-align: center; }

display

Define como um elemento deve ser exibido na página.

Exemplo: .Card {   display: flex; }

padding

Bota espaçamento dentro de um elemento, entre o texto e a borda.

Exemplo: body {   padding: 15px; }

width

Define a largura de um elemento

Exemplo: .card {   width: 300px; }

height

Define a altura de um elemento.

Exemplo: .card {   height: 200px; }

font-family

Define a fonte utilizada no texto.

Exemplo: body { font-family: Arial, sans-serif; }
