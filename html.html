Voltar

<a>

Cria um link para outra página ou recurso.

Exemplo: <a href="https://google.com">Ir para o Google</a>

<code>

Indica um texto dentro da tag é um codigo

Exemplo: <code> <tittle> Document <tittle> <code>

<img>

Insere uma imagem na página.

Exemplo: <img src="imagem.png" alt="Descrição imagem">

<iframe>

Incorpora outra página dentro da página atual.

Exemplo: <iframe src="https://SesiSenai.com.br" ></iframe>

<div>

Define uma divisão no HTML.

Exemplo: <div> outros Código dentro da DIV </div>

<footer>

Faz um rodapé para o conteúdo.

Exemplo: <footer>Conteúdo do rodapé</footer>

<ul>

Faz uma lista não ordenada.

Exemplo: <ul>   <li>Item 2</li> </ul>

<h1>

Faz um título, geralmente o mais importante da página.

Exemplo: <h1>Título da Página</h1>

<link>

Define a relação entre o documento atual o outro usado para linkar o CSS ao html.

Exemplo: <link rel="stylesheet" href="style.css">

<hr>

Insere uma linha.

Exemplo: <p>Texto cima</p> <hr> <p>Texto baixo</p>
