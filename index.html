Bem-vindo ao Glossário de Programação Web!

Você pode alternar entre páginas no canto superior esquerdo.

Este glossário é um trabalho pedido pela professora Senna e foi feito para dar uma força pra quem está começando no mundo da programação. A ideia é explicar, de um jeito simples, os principais termos e conceitos das três linguagens básicas para criar sites: HTML, CSS e JavaScript. Aqui você vai encontrar definições rápidas e diretas, que vão desde coisas bem básicas, como o que é uma tag no HTML, até assuntos um pouco mais avançados, como eventos no JavaScript ou truques de estilização no CSS.
