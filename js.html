console.log

Exibe mensagens no console, usadas geralmente pra debug.

Exemplo: < script > console.log("Você printou uma menssagem!"); < /script >

if

Estrutura condicional que executa o código apenas se uma condição for verdadeira.

Exemplo: < script > let numero = 10; if (numero > 5) { console.log("O número é maior que 5"); } < /script >

else

Usado junto com o if para executar um bloco de código quando a condição é falsa.

Exemplo: < script > let numero = 3; if (numero > 5) { console.log("O número é maior que 5"); } else { console.log("O número é menor ou igual a 5"); } < /script >
